# The_Game_Of_Trusters_Video_Game_Hot_Swappable
A hot swappable video game adaptation of The_Game_Of_Trusters

<h3>Prerequisites</h3>
<ol>
  <li>Type this in terminal: sudo apt-get update</li>
  <li>Type this in terminal: sudo apt-get install ruby-full</li>
  <li>Type this in terminal: sudo apt install liblua5.4-dev</li>
  <li>Type this in terminal and it must be a version that uses liblua5.4: gem install rlua</li>
  <li>Type this in terminal: gem install rerun</li>
</ol>

<h3>How to run the game</h3>
<ol>
  <li>type this in terminal: cd "/path/to/game/directory"</li>
  <li>type this in terminal: rerun ruby The_Game_Of_Trusters_Video_Game_Adaptation_Hot_Swappable.rb</li>
</ol>

<h3>How to run add-on mods</h3>
<ul>
  <li>Add the mod file to your game directory</li>
  <li>The mod must be called this and have the same variables and logic: The_Game_Of_Trusters_Mod.lua</li>
  <li>Only 1 mod file can be loaded at a time</li>
</ul>
